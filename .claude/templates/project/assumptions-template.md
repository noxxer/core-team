## Уровни уверенности

- **High** — есть прямое evidence (метрика, эксперимент, прямое подтверждение от пользователей)
- **Medium** — есть косвенное evidence (аналогия, отраслевое наблюдение, единичные интервью)
- **Low** — есть только гипотеза, нужна валидация перед опорой на это

## Аудит допущений (периодически — раз в месяц / при крупных решениях)

Для каждого high-confidence допущения: подтверждается ли по-прежнему? Появилось ли новое evidence?
Для каждого low-confidence — продвинулись ли в validation plan?

**Trigger переоценки:**
- Возникла tension в Decider — какое допущение под ней?
- Новые данные противоречат существующему допущению
- Decay-review старого DEC: какие из его допущений всё ещё валидны?
