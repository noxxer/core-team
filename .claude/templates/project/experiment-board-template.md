## Принципы

- Гипотеза без метрики — это намерение, не эксперимент
- Без критерия успеха ДО старта — будут споры о результате
- Без срока — running → forever
- Один эксперимент = одна гипотеза. Несколько изменений одновременно → не понять что дало эффект
- Kill criterion должен быть тоже явным (Confirmation Bias защита)
