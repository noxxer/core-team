# Глоссарий проекта

> Локальный словарь проекта — ограниченный контекст (FPF A.1.1).
> Значение термина определяется ЗДЕСЬ, а не "общепринятым" смыслом.
> Все роли предлагают термины. Keeper поддерживает формат. Facilitator модерирует.

## Соглашения

- **Один термин — одно значение** в контексте проекта (без омонимии)
- **Колонка "Не является"** обязательна при риске путаницы с похожим понятием
- **Английский термин** используется в коде, API, именах переменных и файлов
- При изменении значения термина — обновить, не дублировать (VCS хранит историю)
- Термины упорядочены по предметным блокам, не алфавитно
